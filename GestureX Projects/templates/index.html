GestureX

Gesture Based Control System

A hands-free computer interaction using eye gaze & gesture tracking

Project

Project Goal

The aim of GBCS is to allow hands-free interaction with a computer using eye gaze tracking and hand gestures. It is designed for paralysed and physically challenged individuals.

Technologies

  • OpenCV
  • Mediapipe FaceMesh
  • PyAutoGUI
  • Mathematical Regression Mapping

Features

  • OpenCVCursor Control
  • Calibration System
  • Heatmap Tracking
  • Gesture Recognition
  • Clicking Mechanism (Coming Soon)

Benefits

  • Hands-free usage
  • No Need of mouse
  • Accessible for disabled users
  • Medical application
  • Heatmap based Tracking

Future Updates

  • Improve gesture recognition accuracy
  • Add customizable gestures
  • Integrate Multiple Ways for Control System
